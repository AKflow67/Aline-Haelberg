## Déploiement

1. Créer un repo GitHub `aline-haelberg`
2. Pousser tous les fichiers
3. Connecter à Netlify (New site from Git)
4. Formulaire Netlify activé automatiquement (data-netlify="true")
5. Acheter et configurer le domaine dans Netlify DNS
